# S020 — 3-state pill filter for step search

## Summary

Replaced the internal variant grouping (section labels) in StepSearch with an explicit 3‑state pill‑switch filter. Users can now toggle between "All", "Processors", and "Output" to control which step types appear.

## Design

The pill switch sits between the search bar and the results list:

```
  ┌─────────────────────────────────────┐
  │  🔍 Search steps…                    │
  ├─────────────────────────────────────┤
  │                                     │
  │    [  All  ] [Processors] [ Output ]│
  │       └── inline pill in gray bg    │
  ├─────────────────────────────────────┤
  │  watermark-remover          v1.0.0  │
  │  Detect and remove watermarks       │
  │  test-processor            v1.0.0   │
  │  Apply crop, resize...              │
  └─────────────────────────────────────┘
```

- Three radio‑style pills in a `rounded-full` gray container
- Active pill gets `bg-blue-600` + `shadow-sm` for a raised effect
- Inactive pills are `text-gray-400`, `hover:text-white`
- Accessible: `role="radiogroup"` on the container, `role="radio"` with `aria-checked` on each pill
