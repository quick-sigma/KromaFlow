# S018 — ImageExplorer: fullscreen image overlay on thumbnail click

## Summary

Added an `ImageExplorer` component — a fullscreen modal overlay that opens when clicking an image thumbnail (`Miniature`). Supports navigation through multiple images (keyboard + buttons), close on Escape/backdrop click, and i18n translations.

## ImageExplorer features

- **Fullscreen dark overlay** with centered image (max 90vw × 85vh, object-contain)
- **Close button** (×) in top-right corner
- **Backdrop click** closes the overlay
- **Escape key** closes the overlay
- **Previous/Next buttons** (‹ ›) on left/right edges — only shown when multiple images
- **ArrowLeft / ArrowRight** keyboard navigation
- **Wrap-around navigation** — next from last goes to first, prev from first goes to last
- **Caption** showing image name and counter (e.g., "portrait.jpeg (2/3)")
- **i18n support** — all aria-labels use translations (EN/ES)
- **Full accessibility**: `role="dialog"`, `aria-modal="true"`, keyboard navigation

## Integration flow

```
User clicks thumbnail
  → MiniatureImageWithOptions.onView() fires
  → MiniatureList sets selectedImageId
  → ImageExplorer renders with full image list + initialIndex
  → User navigates (arrows, click buttons, or keyboard)
  → User closes (Escape, backdrop click, or close button)
  → MiniatureList clears selectedImageId → ImageExplorer unmounts
```
