# Processing Queue System with WebSocket Status Updates

Date: 2026-07-02

## Summary

Implemented a processing queue system that processes images one at a time. Added WebSocket endpoints for real-time status updates, a global progress bar, and queue-aware miniature states to prevent re-enqueueing.
