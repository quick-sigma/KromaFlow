# S001 — Inicialización del proyecto

## Resumen
Se inicializó el repositorio Git y se crearon los proyectos base (Backend y Frontend).

## Detalle

- **Repositorio Git** iniciado en la raíz del proyecto.
- **Carpeta `journal/`** creada para llevar bitácora del proyecto.
- **Backend/** — Proyecto FastAPI con Python 3.14:
  - Entorno virtual en `.venv/`
  - Dependencias: `fastapi[standard]`, `uvicorn`
  - Archivo `main.py` con endpoint raíz `GET /`
  - Servidor en puerto `55558`
- **Frontend/** — Proyecto con Vite + React + TypeScript + Tailwind CSS v4 + Zustand:
  - Scaffold con `create-vite` (template `react-ts`)
  - Tailwind CSS v4 integrado vía `@tailwindcss/vite`
  - Store de ejemplo con Zustand (`src/stores/counter.ts`)
  - Servidor en puerto `55559`
- **`run.sh`** — Script para levantar ambos servidores y abrir Brave con dos pestañas.
